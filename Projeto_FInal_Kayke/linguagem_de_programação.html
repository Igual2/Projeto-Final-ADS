HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar a estrutura e o conteúdo de páginas da web. É uma linguagem fundamental para a criação de documentos da web e é interpretada pelos navegadores da web.
HTML

/

Aprendizado adquirido

  • Estrutura Básica: html, head, title, body, entre outras.
  • Tags: O HTML utiliza várias tags para marcar e estruturar o conteúdo: 'h1' a 'h6' para cabeçalhos, 'p' para parágrafos, 'a' para links, 'img' para imagens e 'ul' / 'ol' para listas não ordenadas e ordenadas (utilizamos a tag 'li' para separar em tópicos o que está sendo pedido e posto no nas tegas 'li' ou 'ol') e também há a tag 'table' que, como o nome já diz é utilizado para criar tabelas.
  • Atributos: As tags HTML podem ter atributos para fornecer informações adicionais: tag 'a' pode ter o atributo 'href' para especificar o URL do link, e a tag 'img' pode ter o atributo 'src' para definir o caminho da imagem.
  • Para facilitar na organização e estrutura uniforme do html utiliza-se "Estrutura Semântica:" header, section, article, nav, aside, main, figure, footer, a, em, strong, cite, q, time.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a aparência e o layout de documentos HTML (páginas da web). É uma parte fundamental do desenvolvimento web, permitindo controlar o design e a formatação dos elementos HTML. CSS

///

Aprendizado adquirido

  • Seletores: Os seletores são usados para selecionar elementos HTML aos quais você deseja aplicar estilos. Alguns exemplos de seletores comuns são: seletor de elemento (por exemplo, p para parágrafos), seletor de classe (por exemplo, .classe para elementos com uma determinada classe) e seletor de ID (por exemplo, #id para elementos com um ID específico).
  • Propriedades: As propriedades CSS são usadas para definir o estilo dos elementos selecionados. Existem muitas propriedades disponíveis, como cor de fundo (background-color), cor do texto (color), tamanho da fonte (font-size), margens (margin), preenchimento (padding), etc.
  • Valores: As propriedades CSS são definidas usando valores. Por exemplo, você pode definir a cor de fundo de um elemento para vermelho usando o valor red. Existem também valores numéricos (por exemplo, para tamanhos de fonte) e valores de medidas (por exemplo, px para pixels, % para porcentagem).
  • Seletor de descendente: Você pode usar o seletor de descendente para aplicar estilos a elementos HTML que são descendentes de outros elementos. Por exemplo, você pode selecionar todos os elementos 'li' que estão dentro de um elemento 'ul' usando o seletor 'ul' 'li'.
  • Posicionamento: O CSS oferece várias formas de posicionar elementos na página. Alguns métodos comuns incluem posicionamento estático (o padrão), posicionamento relativo (usando propriedades como top, left, right e bottom em relação à posição normal), posicionamento absoluto (usando coordenadas em relação ao elemento pai) e posicionamento fixo (mantendo um elemento fixo na tela enquanto o usuário rola a página).

JavaScript

JavaScript é uma linguagem de programação de alto nível, principalmente usada para criar conteúdo dinâmico em sites. Ele foi originalmente desenvolvido para ser executado no navegador, permitindo interações do usuário, manipulação de elementos da página, animações, validação de formulários e muitas outras funcionalidades. JavaScript

/

Aprendizado adquirido

  • Interatividade: Com o JavaScript, é possível adicionar funcionalidades interativas às páginas da web sem a necessidade de recarregar a página inteira, melhorando a experiência do usuário.
  • Manipulação de objetos em tempo de execução: O JavaScript possui recursos que permitem manipular objetos em tempo real durante a execução do programa, tornando-o flexível e poderoso.
  • Sintaxe semelhante a outras linguagens: A sintaxe do JavaScript é semelhante a outras linguagens de programação, como C++ e Java, facilitando para os desenvolvedores aprenderem e trabalharem com ela.
  • Manipulação de elementos HTML (DOM): O JavaScript permite manipular elementos HTML, como modificar seu conteúdo, estilos, classes e atributos, dinamicamente por meio de scripts.
  • Controle de eventos: Com o JavaScript, é possível controlar eventos, como cliques de botões, movimentos do mouse e teclas pressionadas, para realizar ações específicas.
